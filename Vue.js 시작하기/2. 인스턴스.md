# 인스턴스

## 인스턴스 소개

```vue.js
var vm = new Vue({
    el: '#app',
    data: {
        message : 'hi'
    }
});
```

+ el : 엘리먼트
+ 바디태그안에서 app라는 id라는 가진 태그를 찾아서 인스턴스를 붙이겠다.
  - 붙이는 순간 vue의 속성과 기능들이 유효해 진다.
