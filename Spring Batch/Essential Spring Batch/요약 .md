## 개발환경
1. java 1.8
2. 인텔리제이
3. MySql

## 배치 작업이란?
1. 실시간 작업과 달리 한번에 모아서 처리하는 작업
2. 정산 작업, 메일 쿠폰 발송, 거래 명세서 생성, 추천 시스템 최신화 작업 등등

## 스프링 배치로 작업이 필요한 이유
1. 풍부한 기능
2. 일관성된 코드
3. 기존 서비스가 스프링 프레임워크로 되어 있는 경우 호환 가능
